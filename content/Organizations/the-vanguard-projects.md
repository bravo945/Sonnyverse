# The Vanguard Projects

The Vanguard projects were a set of military projects revolving around using the recent influx in advanced technologies to create new and powerful military inventions for combat in World War 3. The most recognizable projects from this set are [[aics|AICS]], Which used artificial intelligence for land, and the [[SEAL|SEAL]], which created power armor and other utilities for soldiers to gain an upper advantage on the battleground.

> ![[vanguard-project-symbol.png]]


The Vanguard projects started shortly after the [[big-boom|Big Boom]]
